					
					Informations about the Netflix AIO Tool

===================================================================================================================

At first run install.bat | Do not close the terminal, it is important that all requirements are
installed successfully for the Spotify AIO tool to work. After the installation the Netflix AIO Tool is launching
by itsself.

===================================================================================================================

IF YOU DONT LIVE IN GERMANY, PLEASE USE AN VPN AND CONNECT TO AN GERMAN SERVER (otherwise you cant do anything
with the tool)

===================================================================================================================

If you close the Netflix AIO Tool and want to open it again just run start.bat

Please config the config.js file with your Details (important for generating accounts via Discord Bot)
